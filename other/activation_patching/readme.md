This directory consists of the code used for preliminary experiments with activation patching. It was adapted from [emo-llm](https://github.com/aminbana/emo-llm). 

## To run
```
python main.py --prompt_version "P1" --model_index 2 --activation_patching --task_type crowd
[task_type = (llm, crowd, crowd-h, dialogue, twitter)]
```
